# User Acceptance Testing Checklist

## Admin User Tasks
- [ ] Can login to admin console
- [ ] Can view dashboard
- [ ] Can manage users
- [ ] Can manage devices
- [ ] Can manage resources
- [ ] Can create policies
- [ ] Can view sessions
- [ ] Can approve access requests
- [ ] Can view audit logs
- [ ] Can generate reports

## Regular User Tasks
- [ ] Can login
- [ ] Can view own profile
- [ ] Can register device
- [ ] Can request access
- [ ] Can view own sessions
- [ ] Can logout

## Security Officer Tasks
- [ ] Can view security dashboard
- [ ] Can investigate incidents
- [ ] Can revoke sessions
- [ ] Can export audit logs
- [ ] Can view compliance reports

## Business Scenarios
- [ ] New employee onboarding works
- [ ] Employee offboarding works
- [ ] Device replacement works
- [ ] Access revocation works
- [ ] Emergency access works

## Usability
- [ ] Navigation is intuitive
- [ ] Forms are clear
- [ ] Error messages are helpful
- [ ] Loading states are shown
- [ ] Responsive on mobile

## Accessibility
- [ ] Keyboard navigation works
- [ ] Screen reader compatible
- [ ] Color contrast sufficient
- [ ] Focus indicators visible
